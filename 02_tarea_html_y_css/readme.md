# Segundara tarea - HTML y CSS

En esta segunda tarea intentaremos aplicar todo lo visto hasta ahora de
ambos lenguajes. Así que, manos a la obra!

Nota: El sitio puede ser de la temática que prefieran!!

## Consigna:

### Parte HTML

1. Crear un sitio web compuesto por al menos 3 páginas. (index.html, contacto.html, y la tercera es a elección).
2. Las tres páginas deben estar interconectadas por un menú de navegación funcional y compartir un footer que contenga redes sociales, logo y mensajito de copyright.
3. En index.html debe haber al menos 4 secciones bien definidas con titulo y contenido a elección.
4. En la página a elección debe haber como mínimo una galeria de imagenes o tarjetas utlizando el concepto de inline-block que vimos en la última clase.
5. En la página de contacto, crear un formulario de contacto con los campos nombre, email y mensaje.

### Parte CSS

1. Seleccionar una paleta de colores que contenga, negro, blanco y al menos 2 colores a elección. Pueden ser opuestos o complementarios.
2. Investigar como aplicar fuentes externas. Mínimo 2 fuentes, una genérica y una cuyo aspecto esté relacionado con la temática de la página o con el público al que esta orientada.
3. Aplicar todos los conceptos de dimensiones que vimos, incluyendo porcentajes, viewport, REM y EM para que el sitio sea fácilmente mantenible y adaptable. 
4. Investigar como colocar horizontal las opciones del menú de navegación.
5. Las secciones de la página deben tener un color de fondo diferente para que se note donde termina una y arranca la otra. (Pueden investigar el concpeto de relleno para hacer que el contenido de las secciones no quede tan pegado a la sección siguiente. Lo veremos la clase que viene.)
